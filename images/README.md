# Portfolio Images

This directory contains images for the portfolio website.

## Required Images

Please add the following images to this directory:

1. **decision-module.jpg** - Screenshot or thumbnail of "The Decision module" project
2. **on-demand.jpg** - Screenshot or thumbnail of "On-demand" project
3. **trq-dad.jpg** - Screenshot or thumbnail of "TRQ DAD" project
4. **profile.jpg** - Your professional profile photo (recommended: grayscale or desaturated)

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Project images**: Recommended aspect ratio 4:3 (1.33:1), minimum width 800px
- **Profile image**: Recommended minimum 600x800px (portrait orientation)
- **File size**: Optimize images for web (aim for <500KB per image)

## Placeholder Images

Until you add your real images, the portfolio will show broken image placeholders. You can:

1. Add your own images with the exact filenames listed above
2. Use placeholder services like https://placehold.co/ temporarily
3. Export images from your Framer site
